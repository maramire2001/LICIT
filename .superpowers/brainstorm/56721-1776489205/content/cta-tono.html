¿Cuál es el tono del CTA principal?

Este botón es la única acción que importa en la landing. Define cómo percibe el cliente a LICIT-IA desde el primer segundo.

A

"Ver demo en vivo" — Tono actual (abierto)

Cualquiera puede ver la demo sin compromiso. Bajo fricción, alto volumen de curiosos. Funciona bien si el objetivo es crear awareness masivo.


⚡ Ver demo en vivo →


Problema: Para un producto de $12,500/mes, "cualquiera puede entrar" lo hace sentir barato. El Director General que va por un contrato de $20M no quiere la misma herramienta que usa todo el mundo.

B

"Agendar demo privada" — Tono exclusivo (selectivo)

El cliente llena un formulario corto (empresa, industria, volumen de licitaciones al año). Solo reciben acceso quienes califican. Crea percepción de producto de élite.


🔒 Agendar demo privada → Cupo limitado · Solo empresas activas en licitaciones


Ventaja: Auto-califica leads, crea urgencia, justifica el precio premium.
Riesgo: Menor volumen de leads en etapa temprana.

C

Dos CTAs paralelos — Demo pública + Ruta exclusiva

CTA primario grande para el cliente listo: "Agendar demo privada". CTA secundario para el que quiere ver antes de comprometerse: "Ver demo interactiva" (la demo actual que ya existe).


🔒 Agendar demo privada →
o · Ver demo interactiva sin registro


Ventaja: Captura los dos perfiles: el decisor que quiere hablar con alguien y el técnico que quiere explorar solo. No pierdes ningún lead.
